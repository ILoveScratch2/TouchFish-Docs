# 客户端使用教程

本教程介绍如何使用 TouchFish 客户端进行聊天和文件传输。

## 连接服务器

### 首次连接

启动客户端后会看到连接界面

需要填写以下信息：

| 字段 | 说明 | 示例 |
|------|------|------|
| 服务器 IP | 服务器的局域网地址 | `192.168.1.100` |
| 端口 | 服务器监听的端口 | `8080` |
| 用户名 | 您的昵称（聊天室中显示） | `piao` |

::: tip 如何获取服务器信息？
向服务器管理员询问 IP 地址和端口号。
:::

### 连接流程

1. 输入服务器信息和用户名
2. 点击"连接"按钮
3. 等待服务器响应

#### 自动审批模式

如果服务器开启了自动审批，您会立即进入聊天室。

#### 手动审批模式

如果服务器需要手动审批：

- 耐心等待管理员审批
- 审批通过后自动进入聊天室


## 聊天功能

### 发送消息

#### 基本发送

1. 在输入框中输入消息
2. 点击"发送"按钮
3. 消息会发送给所有在线用户
