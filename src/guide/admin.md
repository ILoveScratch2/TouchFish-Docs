# 管理员使用教程

本教程介绍如何作为 TouchFish 管理员管理聊天室。

## 管理员类型

TouchFish 支持两种管理方式：

### 服务器控制台管理

- **位置**：服务器端命令行
- **权限**：完全控制
- **适用**：服务器所在机器
- **推荐**：主要管理方式

### 远程管理员客户端

- **位置**：任何已授权的机器
- **权限**：大部分管理功能
- **适用**：远程管理
- **推荐**：辅助管理方式

---

## 服务器控制台管理

### 启动服务器

```bash
# Windows
chat.exe

# Linux/macOS
./chat
# 或
python chat.py
```

启动后，您会看到提示符：

```text
>
```

在这里可以输入管理命令。


### 可用命令

管理员客户端支持以下命令：

| 命令 | 功能 | 示例 |
|------|------|------|
| `broadcast` | 广播消息 | `broadcast 通知内容` |
| `ban` | 封禁管理 | `ban ip 192.168.1.110` |
| `enable` | 解除禁用 | `enable ip 114.51.4.1` |
| `set` | 设置配置 | `set EAP on` |
| `accept` | 接受请求 | `accept 0` |
| `reject` | 拒绝请求 | `reject 0` |
| `search` | 搜索用户 | `search online` |
| `req` | 查看请求 | `req` |
| `exit` | 退出 | `exit` |

::: info 权限说明
管理员客户端**不支持**以下命令：
- `admin`（管理员管理）
- `exit`（关闭服务器，只能退出客户端）
- 直接修改配置文件
:::

所有管理员命令都可在服务器控制台执行

### 执行系统命令

在管理员客户端可以执行系统命令：

```bash
# Windows
> cmd ipconfig

# Linux/macOS
> cmd ls -la
```

::: danger 安全警告
执行系统命令时要谨慎，避免执行危险命令。
:::

## 日志查看

### 实时查看日志

服务器控制台会实时显示所有活动。

### 查看日志文件

日志保存在服务器运行目录的 `log.txt` 文件中。
