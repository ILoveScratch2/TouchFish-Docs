# TouchFish V5 开发计划

欢迎了解 TouchFish V5！这是 TouchFish 项目的下一个重大版本。

## 开发状态

TouchFish V5 已经启动开发！我们正在积极开发新功能和改进。

## 新特性

TouchFish V5 将带来诸多激动人心的新特性：

### 核心改进
- **全新架构** - 采用更现代的 Python Flask 技术栈，提升性能和稳定性
- **更好的用户体验** - 基于 Flutter （[TouchFish Astra](https://github.com/ILoveScratch2/TouchFish-Astra) 已迁移为 [TouchFish Client](https://github.com/ILoveScratch2/TouchFish-Client)）重新设计的界面，更加流畅和直观
- **增强的安全性** - 更强的加密和隐私保护，保障数据安全
- **聊天记录同步** - 支持多设备聊天记录同步，随时随地访问历史消息
- **聊天系统增强** - 支持群组、私聊等多种聊天方式，添加公告、论坛等功能

### 功能增强
- **多平台支持** - 由 [TouchFish Client](https://github.com/ILoveScratch2/TouchFish-Client) 提供的基于 Flutter 的完善的跨平台体验
- **富文本消息** - 更丰富的消息格式，可预览
- **文件传输优化** - 更快、更稳定的传输
- **群组和私聊** - 增强的群聊体验
