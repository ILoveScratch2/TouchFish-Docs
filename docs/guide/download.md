# 下载 TouchFish

欢迎下载 TouchFish！根据您的需求和平台，您可以选择不同的发行版本。

## 版本选择指南

> [!WARNING]
> 大部分发行版不支持 TouchFish V4 协议，仅支持 TouchFish V3 及更早版本。

TouchFish 拥有多个发行版本，各有特色：

- **LTS** —— 官方根版本，功能完善，稳定可靠。
- **Astra** —— 全平台支持，UI 精美，支持 Markdown 和 LaTeX。
- **More (Lite)** —— 更好的性能，更快的下载速度，**完全兼容 TouchFish V4**。
- **UI Remake (TFUR)** *(不支持 TouchFish V4)* —— 现代化 UI，支持 Markdown、代码高亮和部分 LaTeX。
- **Pro** *(不支持 TouchFish V4)* —— 支持 Markdown，提供基本的 LaTeX 支持和用户高亮功能。
- **Plus** *(不支持 TouchFish V4)* —— 功能扩展版本（不兼容其他版本）。
- **For Android** *(不支持 TouchFish V4)* —— 安卓移动端（已不推荐使用）。

## 相关软件

- **Cloud Studio Chat (CSC)** —— 使用 C++ 编写的高性能聊天软件，兼容性强（不兼容 TouchFish 协议）。
- **TouchMouse** —— 基于 TouchFish 的聊天项目，优化了 GUI 并添加了大量功能（不兼容 TouchFish 协议）。

## 各版本下载

### LTS（官方版，包含 CLI 版本）

**支持平台**：Windows、macOS、Linux（需图形界面）

**特点**：根版本，功能完善，与兼容版本互通。

**下载地址**：
- [GitHub Releases](https://github.com/TouchFishGJ/TouchFish/releases)
- [镜像站](https://mirror.ilovescratch.us.ci/TouchFish/LTS/)

**包含组件**：LTS.py / LTS.exe / LTS（服务器/客户端二合一）

详细说明请参阅 [LTS 版本说明 Wiki](https://github.com/TouchFishGJ/TouchFish/wiki/LTS) 及站点其他文档。

---

### Astra

**支持平台**：Windows、macOS、Linux（需图形界面）、Android、iOS、Web

**特点**：全平台兼容，UI 精美，支持 Markdown 和 LaTeX。

**下载地址**：
- [GitHub Releases](https://github.com/TouchFishGJ/TouchFish-Astra/releases)
- [镜像站](https://mirror.ilovescratch.us.ci/TouchFish/Astra)

**包含组件**：仅客户端。

---

### More (Lite)

**支持平台**：Windows、macOS、Linux（需图形界面）

**特点**：更好的性能，更快的下载速度，**完全兼容 TouchFish V4 协议**。

**下载地址**：
- [Gitee 仓库](https://gitee.com/TouchFishGJ/TouchFish-More)
- [镜像站](https://mirror.ilovescratch.us.ci/TouchFish/More(Lite)/)

**特点**：现代化 UI，支持 Markdown、代码高亮和部分 LaTeX。

**下载地址**：
- [GitHub Releases](https://github.com/TouchFishGJ/TFUI-Remake/releases)

> [!WARNING]
> 不支持 TouchFish V4 协议。

**支持平台**：Android

**特点**：移动端支持，但容易被国产安卓系统杀进程。

**下载地址**：无

**重要提示**：此版本已不推荐使用，建议改用 Astra 版本的 Android 客户端。

## 版本兼容性说明

| 兼容类型               | 版本                                                         |
| ---------------------- | ------------------------------------------------------------ |
| 兼容 V3 协议           | LTS、CLI、UI Remake、Pro、More、For Android、Astra           |
| 兼容 V4 协议           | LTS、Astra、More                                             |
| 不兼容 V3 协议         | Plus                                                         |
| 不兼容（相关软件）     | Cloud Studio Chat (CSC)、TouchMouse                          |

- 上述兼容版本之间可以互相连接，服务器端和客户端可使用不同的兼容版本。
- Plus 版本不与其他版本兼容，且不同 Plus 版本之间也不兼容。
- 相关软件（CSC、TouchMouse）不与 TouchFish 主版本兼容。
