为什么选择 TouchFish？

专注局域网通讯场景，提供简洁高效的聊天体验

无需公网

完全基于局域网工作，断网环境照样畅聊，机房、内网的理想选择

轻量快速

体积小巧，资源占用低，启动即用，无复杂配置

跨平台

多种平台支持，想在哪用就在哪用

易于部署

一台电脑做服务器，其他人连接即用，3 分钟搞定

管理便捷

独立的管理端，支持用户管理、消息监控等多种控制功能

开源免费

完全开源，社区驱动，欢迎参与贡献

发行版

丰富的发行版本，总有一款适合你（详见 下载 页面）

LTS (GUI)

官方根版本，稳定可靠，由 @2044-space-elevator 维护。支持 Windows/MacOS/Linux，适合大多数用户

CLI 版本 (LTS内置)

纯命令行界面，适合无 GUI 的 Linux 服务器环境，随LTS发行

UI Remake

现代化界面设计，支持 Markdown、代码高亮、部分 LaTeX。基于 Node.js

Astra

精美 UI，全平台兼容。支持 Markdown 和 LaTeX

Pro 版

功能增强，Markdown、LaTeX、用户高亮一应俱全

More(Lite) 版

性能优化版，更快的下载速度

Plus 版

功能增强版，拥有更多特性（不兼容）
